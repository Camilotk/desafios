## Minha versão

Fiz de forma recursiva pra fazer os numerozinhos mas ai quando teve que lidar com o big int desgraçou de vez e inventei de separar o numero chunks.

Precisa do [node instalado](https://nodejs.org/en/)!!

Como rodar:
```bash
# Instalar Yarn
npm i -g yarn

# Baixar dependencias
yarn

# Rodar
yarn start
```
